# Operação

## Execução Local

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Build e Preview

```bash
npm run build
npm run preview
```

## Checklist Operacional

- Confirmar aceleração de hardware habilitada no navegador.
- Executar `npm run typecheck` antes de deploy.
- Validar interações de mouse e touch.
- Verificar FPS sob presets mais pesados.
