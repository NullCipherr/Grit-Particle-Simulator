# Security Policy

## Supported Versions

Security fixes are provided on a best-effort basis for the latest `main` branch state.

## Reporting a Vulnerability

Please do **not** open a public issue for security vulnerabilities.

Report privately to: `nullcipherr@proton.me`

Include:

- a clear vulnerability description
- reproduction steps
- potential impact
- suggested mitigation (if available)

## Response Process

- Initial acknowledgement target: 72 hours
- Triage and impact validation
- Fix development and verification
- Coordinated disclosure after patch release
